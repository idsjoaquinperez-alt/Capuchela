# Capuchela - Cerveza Artesanal

Este es el proyecto final para el curso de Desarrollo Web de Coderhouse.

## Autor
Joaquín Pérez

## Descripción
Sitio web para "Capuchela", una marca de cerveza artesanal inspirada en la tradición de abadía y el batch pequeño.
El proyecto incluye:
- Página de Inicio
- Conocenos (Historia)
- Estilos (Catálogo de productos)
- Blog (Novedades)
- Contacto (Formulario)

## Tecnologías
- HTML5 (Semántico)
- CSS3 (Flexbox, Grid, Responsive Design)
- SASS (Preprocesador CSS)
- Bootstrap 5 (Componentes como Navbar y Carousel)

## Estructura
El proyecto utiliza una arquitectura SCSS modular (`base`, `layout`, `pages`, `utilities`) compilada manualmente.
